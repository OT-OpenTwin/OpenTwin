Collection of changes for the next release

Features/Changes
- Improved the storage and access of complex numbers
- Changed curve/plot concept
- Curves have now properties to define a generic query on the datalake
- Plots have now the option to select secondary parameter that are not used for partitioning the curves
- Queries support now regex expressions
- Added a 'datetime' type to parameterized data that auto-detects timestamps or time durations, which are then stored into the database as milliseconds.
- Introduced a library for Python Scripts
- Scripts can now be imported by selecting required property
