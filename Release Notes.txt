Collection of changes for the next release

Features/Changes
- Data processing now follows the solver concept, i.e. pipeline design is separate from solver with run settings
- Multiple data pipelines are now possible
- Renaming of graphics scenes is now possible
- Snapping and unsnapping of connections is now possible
- Multi drag/drop of blocks is now handled in one state
- Python environments were added
- Python packages are automatically installed, either per selected environment description or import statements in the executed script
- Improved visual feedback for importing large numbers of files
- Improved speed of data transport in pipeline by factor ~3
